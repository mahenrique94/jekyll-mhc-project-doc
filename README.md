# mhc-project-doc

## Instalação

Adicione esse linha no `Gemfile` do seu site feito com Jekyll:

```ruby
gem "mhc-project-doc"
```

Adicione essa linha no `_config.yml` do seu site fieto com Jekyll:

```yaml
theme: mhc-project-doc
```

Então execute:

    $ bundle

Ou instale você mesmo com:

    $ gem install mhc-project-doc
